# Hope Channel Southeast Asia Logo

To add the Hope Channel Southeast Asia logo to the application:

1. Create or obtain a PNG file named `hope_channel_logo.png`
2. Place it in this directory: `assets/images/`
3. Make sure the image dimensions are approximately 200x200 pixels to 400x400 pixels
4. Ensure the image has a transparent background if possible

Once you've added the logo file, run:
```bash
flutter pub get
```

Then restart the application to see the logo appear next to the organization name.
